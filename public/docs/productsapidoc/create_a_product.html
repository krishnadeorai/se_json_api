ProductsApiDoc API

Create a product

POST /products

Parameters

Name Description
type Should always be set to <strong>products</strong>.
attributes[name] Name of product.
attributes[category] Category of product.
attributes[sold-out] Product is out of stock or not. Enter value as 0 or 1. 0 means product is available and 1 means product is not available.
attributes[price] Price of product.
attributes[under-sale] Product is under sale or not. Enter value as 0 or 1. 0 means product is not under sale and 1 means product is under sale.
attributes[sale-price] Selling Price of product. Required when under-sale value is 1.
attributes[sale-text] Offer on product. Required when under-sale value is 1.

Request

Headers

Content-Type: application/vnd.api+json
Host: example.org
Cookie:

Route

POST /products

Body

{"data":{"type":"products","attributes":{"name":"Test","category":"Makeup","sold-out":0,"price":1000,"under-sale":1,"sale-price":500,"sale-text":"50% Off"}}}

Response

Headers

Content-Type: application/vnd.api+json

Status

201 Created

Body

{"data":{"id":"49","type":"products","links":{"self":"http://example.org/products/49"},"attributes":{"name":"Test","category":"Makeup","sold-out":false,"under-sale":true,"price":1000,"sale-price":500,"sale-text":"50% Off"}}}
