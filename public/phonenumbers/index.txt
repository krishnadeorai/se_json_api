Create a phone_number
---------------------

Parameters:
  * type -       Should always be set to <strong>phone_number</strong>.

  * name -       Name of group of which phone number belongs

  * mobile -       Mobile No

  * contact -       The contacts assigned to the phone_number.


Request:
  POST /phone-numbers
  Content-Type: application/vnd.api+json
  Cookie: 
  Host: example.org

  {"data":{"type":"phone_numbers","attributes":{"name":"home","mobile":"1231231231"},"relationships":{"contact":{"data":{"id":1135,"type":"contacts"}}}}}

Response:
  Status: 201 Created
  Content-Type: application/vnd.api+json

  {"data":{"id":"164","type":"phone-numbers","links":{"self":"http://example.org/phone-numbers/164"},"attributes":{"name":"home","mobile":"1231231231"},"relationships":{"contact":{"links":{"self":"http://example.org/phone-numbers/164/relationships/contact","related":"http://example.org/phone-numbers/164/contact"}}}}}
